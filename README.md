# Natural Craft storefront

This workspace now contains a responsive multi-page Natural Craft promotion storefront with official site imagery integrated into the design.

## Pages

- `index.html` - homepage and brand promotion landing page
- `collections.html` - filterable Natural Craft collection view
- `product.html?slug=dune-bloom-napkin-set` - product detail template
- `cart.html` - persistent cart view
- `checkout.html` - demo checkout form and local confirmation flow
- `story.html` - Natural Craft brand story page

## Assets

- Official Natural Craft images are stored in `assets/natural-craft/`.
- Shared storefront logic lives in `site.js`.
- Shared styling lives in `styles.css`.

## Run locally

Open `index.html` directly in a browser, or serve the folder with a simple static server.

Example:

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000/index.html`.
